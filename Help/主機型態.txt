【主機型態】
--------------------------------------------------------
1.實體設備 = 主機設備 + 非主機設備
  a.實體設備：全部實體設備資料
  b.主機設備：系統設備 + 網路設備
  c.迴路設備：系統設備 + 網路設備 + 環境設備 + 週邊設備

2.主機設備 = 作業主機 + 空機
  a.作業主機：有安裝作業系統的主機
  b.空機：未安裝作業主機的主機設備

3.作業主機 = 系統主機 + 網路主機
  a.系統主機：設備型態為系統設備的作業主機
  b.網路主機：設備型態為網路設備的作業主機

4.實體設備 = 中心設備 + 代管設備
  a.中心設備：設備維護人員為資訊中心成員的設備
  b.代管設備：設備維護人員非資訊中心成員的設備

5.作業主機 = 中心主機 + 代管主機
  a.中心主機：作業維護人員為資訊中心成員的作業主機
  b.代管主機：作業維護人員非資訊中心成員的作業主機

6.作業主機 = 個人主機 + 伺服主機
  a.個人主機：安裝個人作業系統的作業主機
  b.伺服主機：安裝非個人作業系統的作業主機

7.作業主機 = 實體主機 + 虛擬主機
  a.實體主機：設備種類非虛擬主機的作業主機
  b.虛擬主機：設備種類為虛擬主機的作業主機

8.作業主機 = 功能主機 + 非功能主機
  a.功能主機：有安裝系統功能的主機
  b.非功能主機：未安裝系統功能的主機

9.作業主機 = 單一主機 + 多重主機 + 虛擬主機
  a.單一主機：同一主機設備有一個作業主機
  b.多重主機：同一主機設備有多個作業主機(虛擬主機除外)
